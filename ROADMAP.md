# G. Learning Roadmap

## Phase 1: Web Fundamentals
- **HTML/CSS**: Flexbox, Grid, Responsive Design.
- **JavaScript**: ES6+, `fetch()` API, DOM manipulation, JSON handling.
- **HTTP**: Methods (GET, POST), Headers, Status Codes (200, 404, 500), JSON structure.

## Phase 2: Python & Django Basics
- **Python**: Lists, Dictionaries, Classes, Virtual Environments (`venv`), `pip`.
- **Django**: `startproject`, `startapp`, MTV Architecture (Model-Template-View), Routing (`urls.py`), Template Tags (`{% if %}`).

## Phase 3: REST API & FastAPI
- **REST Concepts**: Statelessness, Endpoints.
- **FastAPI**:
    -   Pydantic Models (Data validation).
    -   Dependency Injection.
    -   Async/Await (Asynchronous programming).
    -   Swagger UI (`/docs`).

## Phase 4: LLM & RAG Concepts
- **Embeddings**: Converting text to numbers. Why semantic search works.
- **Vector Databases**: How Nearest Neighbor (KNN/ANN) search works.
- **RAG Architecture**: The concept of "Context Window" and injecting data into prompts.
- **Prompts**: Prompt Engineering basics (System Implementation, Context delimiters).

## Phase 5: LangChain Deep Dive
- **Chains**: `LCEL` (LangChain Expression Language).
- **Retrievers**: Simularity search, MMR (Maximal Marginal Relevance).
- **Loaders**: `DocumentLoader` for PDF, Text, CSV.

## Phase 6: Deployment & Scaling (Future)
- **Docker**: Containerizing your applications.
- **Cloud**: AWS/GCP basic concepts.
- **Optimization**: Caching embeddings, hybrid search (Keyword + Vector).
